Čo je SVM?

Support Vector Machine (SVM) je algoritmus supervised learning, ktorý hľadá optimálnu nadrovinu (hyperplane) na oddelenie dát do tried. SVM maximalizuje margin – vzdialenosť medzi nadrovinou a najbližšími dátovými bodmi z oboch tried. Tieto kľúčové body sa nazývajú podporné vektory.

Matematická formulácia

Pre lineárne separovateľné dáta hľadáme nadrovinu:

f(x) = w · x + b = 0

Optimalizačný problém (primal):

min ½ ||w||²
s.t. yᵢ(w · xᵢ + b) ≥ 1, ∀i

Pre nelineárne separovateľné dáta sa pridáva slack premenná ξᵢ a parameter C (regularizácia), čím vzniká soft-margin SVM:

min ½ ||w||² + C · Σ ξᵢ
s.t. yᵢ(w · xᵢ + b) ≥ 1 - ξᵢ, ξᵢ ≥ 0

Duálna formulácia a kernel trick

Cez Lagrangeove multiplikátory αᵢ sa problém prevádza do duálnej formy, kde sa objavuje iba skalárny súčin xᵢ · xⱼ. Ten môžeme nahradiť jadrovou funkciou K(xᵢ, xⱼ):

max Σ αᵢ - ½ ΣΣ αᵢ αⱼ yᵢ yⱼ K(xᵢ, xⱼ)
s.t. 0 ≤ αᵢ ≤ C, Σ αᵢ yᵢ = 0

Kernel trick umožňuje implicitne mapovať dáta do vyššej dimenzie bez explicitného výpočtu transformácie φ(x), keďže K(xᵢ, xⱼ) = φ(xᵢ) · φ(xⱼ).

Kedy použiť SVM?

✓ Výhody

  • Efektívny vo vysokodimenzionálnych priestoroch
  • Funguje aj keď dimenzií > vzoriek
  • Pamäťovo úsporný (len podporné vektory)
  • Flexibilný vďaka jadrovým funkciám
  • Robustný voči overfittingu (max margin)

✗ Nevýhody

  • Pomalý pri veľkých datasetoch O(n²–n³)
  • Citlivý na výber jadra a parametrov
  • Nedáva priamo pravdepodobnosti
  • Náročný na interpretáciu
  • Nefunguje dobre pri veľa šume

Jadrové funkcie (Kernels)

Jadrové funkcie transformujú dáta do vyššieho dimenzionálneho priestoru, kde sa stávajú lineárne separovateľnými. Kľúčová výhoda: nepotrebujeme explicitne vypočítať túto transformáciu.

Lineárne

K(x,y) = x · y

Pre lineárne separovateľné dáta. Bez transformácie do vyššej dimenzie.

RBF

K(x,y) = exp(-γ||x-y||²)

Gaussovo jadro. Mapuje do nekonečno-dimenzionálneho priestoru.

Polynomiálne

K(x,y) = (x·y + c)ᵈ

Stupeň d určuje komplexnosť rozhodovacej hranice.

Lineárne jadro – podrobne

Najjednoduchšie jadro, zodpovedá klasickej lineárnej klasifikácii. Rozhodovacia hranica je vždy priamka (2D), rovina (3D) alebo nadrovina (vyššie dimenzie). Výhodou je rýchlosť a interpretovateľnosť – váhový vektor w priamo ukazuje dôležitosť jednotlivých príznakov.

K(xᵢ, xⱼ) = xᵢᵀ · xⱼ

Použitie: textová klasifikácia, vysokodimenzionálne riedke dáta, lineárne separovateľné problémy.

RBF (Gaussovo) jadro – podrobne

Najpoužívanejšie nelineárne jadro. Parameter γ (gamma) kontroluje "dosah" každého bodu. Vysoké γ = úzky dosah (riziko overfittingu), nízke γ = široký dosah (riziko underfittingu). RBF implicitne mapuje do nekonečno-dimenzionálneho priestoru (Taylorov rozvoj exp funkcie).

K(xᵢ, xⱼ) = exp(-γ · ||xᵢ - xⱼ||²)

Použitie: väčšina nelineárnych problémov, predvolené jadro v mnohých knižniciach.

Polynomiálne jadro – podrobne

Mapuje dáta do priestoru polynomiálnych kombinácií príznakov. Stupeň d kontroluje komplexnosť. d=1 → lineárne, d=2 → kvadratické (elipsy, paraboly), d=3 → kubické hranice. Parameter c posúva dôležitosť vyššieho vs. nižšieho rádu.

K(xᵢ, xⱼ) = (γ · xᵢᵀ · xⱼ + c)ᵈ

Použitie: spracovanie obrazov, prírodné jazyky, problémy s polynomiálnymi vzťahmi.
