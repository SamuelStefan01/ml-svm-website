# Support Vector Machine (SVM)

Interactive web application implementing a Support Vector Machine
from scratch in pure JavaScript (without ML libraries).

## Features
- Custom implementation of the SMO algorithm
- Linear, RBF, and polynomial kernels
- Interactive canvas – click to add points and observe the decision boundary
- k-fold cross-validation, ROC curve, and Grid Search heatmap
- Learning curve with automatic interpretation
- Comparison: SVM vs k-NN vs Logistic Regression vs Decision Tree

## Running the Application
Simply open `svm_final.html` in a web browser – no installation required.

# Metóda podporných vektorov (SVM)

Interaktívna webová aplikácia implementujúca Support Vector Machine
od základov v čistom JavaScripte (bez ML knižníc).

## Funkcie
- Vlastná implementácia SMO algoritmu
- Lineárne, RBF a polynomiálne jadro
- Interaktívny canvas – klikaj body, sleduj decision boundary
- k-fold krížová validácia, ROC krivka, Grid Search heatmapa
- Krivka učenia s automatickou interpretáciou
- Porovnanie: SVM vs k-NN vs Logistická regresia vs Rozhodovací strom

## Spustenie
Stačí otvoriť `svm_final.html` v prehliadači – žiadna inštalácia.
